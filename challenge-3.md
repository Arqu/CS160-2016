# Challenge 3 - Palindromization

Given a string `x` on stdin, output to stdout the largest substring that is a palindrome. If there are multiple print out the first one.

	Example:
	> abcdedcba
	> abcdedcba

	> aaaabcccccb
	> bcccccb

	> a
	> a

	> abcdabcdzzzzzzdcbagggg
	> abcdzzzzzzdcba

	> aaabbb
	> aaa
